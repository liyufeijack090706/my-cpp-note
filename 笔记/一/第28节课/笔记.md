# string牛逼！（破音

## 赋值
~~~c++
string s1 = "hello";
string s2 = "world";
~~~
## 求长度
~~~c++
cout << s1.length() << endl;
cout << s2.size() << endl;
~~~
## 拼接
~~~c++
string s3  = s1+s2;
cout << s3 << endl;
~~~
## 赋值
~~~c++
s3 = s2;
cout << s3 << endl;
~~~
## 比大小
~~~c++
cout << (s1>s3) << endl;//字典序
~~~
